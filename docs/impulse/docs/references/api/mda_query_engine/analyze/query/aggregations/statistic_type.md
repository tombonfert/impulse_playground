StatisticType enum for defining supported statistics types.


## StatisticType

```python
class StatisticType(Enum)
```

Enumeration of supported statistic types for aggregations.

**Arguments**:

- `MIN` (`str`): Minimum value statistic.
- `MAX` (`str`): Maximum value statistic.
- `MEAN` (`str`): Mean (average) value statistic.
- `MEDIAN` (`str`): Median value statistic.
